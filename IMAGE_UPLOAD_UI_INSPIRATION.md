## 🔍 **Specific UI Patterns to Look For**

### **Pattern 1: Bottom Sheet (Mobile)**
- Camera/Gallery options slide up from bottom
- Common on iOS/Android
- **Example**: Instagram, WhatsApp profile picture

### **Pattern 2: Modal with Tabs**
- Tabs for Camera/Gallery
- Clean, organized
- **Example**: Facebook profile picture

### **Pattern 3: Inline Cropper**
- Cropper appears in same view
- No modal overlay
- **Example**: Twitter profile picture

### **Pattern 4: Drag & Drop**
- Drag image onto avatar
- Visual feedback
- **Example**: Slack profile picture

### **Pattern 5: Step-by-Step Wizard**
- Step 1: Select source
- Step 2: Crop/Edit
- Step 3: Preview & Confirm
- **Example**: LinkedIn profile picture

## 🎯 **What to Look For**

When browsing these resources, pay attention to:

1. **User Flow**: How smooth is the process?
2. **Mobile Experience**: Does it work well on phones?
3. **Visual Feedback**: Clear progress indicators?
4. **Error Handling**: How are errors shown?
5. **Accessibility**: Easy to use for everyone?
6. **Performance**: Fast and responsive?
