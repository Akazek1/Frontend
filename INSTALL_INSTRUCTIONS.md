# Installation Instructions for Image Cropper

## Required Packages

You need to install the following packages for the image cropper to work:

```bash
cd HWA_Frontend
npm install react-easy-crop @radix-ui/react-slider
```

Or if using yarn:
```bash
yarn add react-easy-crop @radix-ui/react-slider
```

Or if using pnpm:
```bash
pnpm add react-easy-crop @radix-ui/react-slider
```

## After Installation

1. Restart your dev server
2. The image cropper should now work when you select a profile picture

## Features

- ✅ Image preview before upload
- ✅ Crop and center image
- ✅ Zoom control (1x - 3x)
- ✅ Rotation control (0° - 360°)
- ✅ Round crop shape for profile pictures
- ✅ Save or cancel options
