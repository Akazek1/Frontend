# Akazek Frontend App

Next.js 15 consumer application for the Akazek marketplace.

## Quick Start

```bash
npm install
cp .env.local.example .env.local
# Edit .env.local with API URL
npm run dev
```

**Frontend runs on:** `http://localhost:3000` (or next available port)
