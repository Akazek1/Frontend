# Marketing site images

Drop image files here with these exact names and they appear automatically on
`/welcome`. Until a file exists, the site shows a labelled placeholder (nothing
breaks).

| File | Where it shows | What to use | Recommended |
|------|----------------|-------------|-------------|
| `app-home.png` | Hero — inside the phone frame | The app home screen ("Good Afternoon…") | Portrait screenshot, PNG |
| `for-workers.jpg` | "For workers" card | A job-posting card (what a worker browses) | Landscape, JPG |
| `for-employers.jpg` | "For employers" card | A worker-profile card (what an employer browses) | Landscape, JPG |

Tips:
- Keep the same filename when you replace an image; no code change needed.
- Compress to < ~400 KB each so the page stays fast.
- Get permission before publishing anyone's photo.
